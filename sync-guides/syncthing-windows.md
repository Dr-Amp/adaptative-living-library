# Syncthing Windows

Install Syncthing, share the library mirror folder, then open it in Obsidian. Example path: `C:\Users\<you>\Syncthing\Living-Ops-Library`.

## Safety

- Do not expose Syncthing GUI publicly.
- Do not sync secrets or `.env` files.
- Write human notes into `_INBOX_USER/`.
- Let agents promote reviewed material into canon.
