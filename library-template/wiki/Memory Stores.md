# Memory Stores

Use separate stores for separate jobs:

- **Living Library**: durable, human-readable canon.
- **Session search**: transcript history and evidence lookup.
- **Active memory**: compact durable facts; write only with clear value.
- **Task systems**: current work, not long-term truth.
- **Raw/inbox**: unreviewed evidence.

Rule: do not let raw logs become canon without Librero review.
