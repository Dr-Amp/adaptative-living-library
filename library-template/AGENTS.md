# AGENTS — Living Ops Library

## Role

This library is the curated knowledge layer for a Hermes agent team.

## Writer policy

- One writer at a time.
- Raw sources and inbox notes are evidence, not canon.
- Every important change gets a short log entry.
- Do not create pages to appear busy.

## Default answer policy

When answering from this library:

1. run or reason through the cheap preflight;
2. read the smallest relevant canon pages;
3. cite local paths;
4. separate evidence from inference;
5. say what is not covered.

## Prohibited by default

- No service restarts.
- No cron creation.
- No model/provider changes.
- No active memory writes.
- No credential edits.
- No external publishing.
